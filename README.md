Geospatial Analysis of Seller Payments – Brazilian E-commerce Store
Project Overview

This project involves creating an interactive Tableau dashboard visualizing Brazilian e-commerce sellers based on geolocation, with bubble sizes representing total payments received from customers.

The dashboard helps to:

Track sales performance by region

Identify high-revenue sellers and regions

Support data-driven allocation and strategy decisions

Click here

Data Source

Kaggle – Brazilian E-Commerce Public Dataset (Olist)

Contains seller information, geolocation, orders, and payment details.

Data was cleaned and transformed before visualization in Tableau.

Tools & Skills

Tableau: Maps, Bubbles, Filters, Dashboards

Excel: Data validation and preprocessing

Data Analytics & Visualization: Geospatial analysis, business reporting

Files & Structure

data/: Dataset containing seller payments and geolocations

tableau_dashboard/: Tableau dashboard file (.twbx)

README.md: Project documentation

How to Use

Open the Tableau dashboard (SellerPaymentsDashboard.twbx) in Tableau Desktop or Tableau Public.

Explore interactive maps showing seller payments by location.

Use filters to analyze regions, seller names, and total payments.

Key Outcomes

Identified top-revenue regions and sellers in the Brazilian e-commerce market.

Created an interactive dashboard that supports strategic business decision-making.

Practiced geospatial data visualization and analytics skills in Tableau.
