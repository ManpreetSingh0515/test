## Project Overview
This project involves creating an **interactive Tableau dashboard** visualizing Brazilian e-commerce sellers based on geolocation, with bubble sizes representing total payments received from customers. 

The dashboard helps to:  
- Track sales performance by region  
- Identify high-revenue sellers and regions  
- Support data-driven allocation and strategy decisions  

**Live Dashboard (Tableau Public):** [Click here](https://public.tableau.com/app/profile/manpreet.kamaljeet.singh.sahota/viz/BrazilianE-commercestoretotalpaymentanalysisusinggeolocation/Sheet1)

## How to Use
1. Open the Tableau dashboard (`SellerPaymentsDashboard.twbx`) in Tableau Desktop or Tableau Public.  
2. Explore interactive maps showing **seller payments by location**.  
3. Use filters to analyze **regions, seller names, and total payments**.  
4. Refer to `scripts/data_cleaning.py` to see how data was processed before visualization.
